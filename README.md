# clases-de-IA-etica

## Curso de IA

Repositorio maestro privado.

## Descripción

Este repositorio contiene las clases del curso de IA, organizadas por módulos semanales.  
Cada clase está pensada para subirse de forma ordenada, con su propio archivo `index.html` y los recursos necesarios dentro de carpetas limpias y fáciles de mantener.

## Estructura

- `index.html` portal general.
- `clase-0/`
- `clase-1/`
- `clase-2/`
- `clase-3/`
- `clase-4/`
- `clase-5/`
- `clase-6/`
- `clase-7/`
- `clase-8/`
- `clase-9/`
- `clase-10/`
- `assets/` para logo e imágenes.

## Flujo

Sube solo la clase vigente cada lunes.

## Notas

- Usa nombres en minúscula.
- Evita espacios en archivos y carpetas.
- Cada clase debe tener su propio `index.html`.
- Mantén actualizados los enlaces internos del portal.

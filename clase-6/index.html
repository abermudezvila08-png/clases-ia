Clase 6: Regresión y Clasificación con Scikit-Learn

Tu primer modelo práctico para predecir números y categorías.

Objetivos

  • Diferenciar regresión y clasificación.
  • Cargar un dataset y dividirlo en entrenamiento y prueba.
  • Entrenar regresión lineal y regresión logística.
  • Evaluar con MSE y accuracy.
  • Interpretar si el modelo es confiable o no.

Regresión vs clasificación

TipoObjetivoEjemploMétrica
RegresiónPredecir un valor numérico.Precio de una casa.MSE.
ClasificaciónPredecir una categoría.Spam / No spam.Accuracy.

Scikit-Learn

Es una biblioteca de Python muy usada para hacer Machine Learning de forma sencilla. Te permite cargar datos, dividirlos, entrenar modelos y medir resultados con pocas líneas de código.

Reglas importantes

  • No uses datos de prueba para entrenar.
  • Divide entrenamiento y prueba antes de ajustar el modelo.
  • No te cases con un solo número de evaluación.
  • Ten cuidado con el desbalanceo de clases.

Práctica

Ejecuta un ejemplo en Colab, cambia el dataset o prueba otro modelo simple, y anota el resultado.

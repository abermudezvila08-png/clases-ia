Clase 4: Python básico para IA

Primer contacto con el lenguaje que usarás para trabajar en Colab.

Objetivos

  • Entender qué es Python y para qué sirve.
  • Ver variables, listas y condiciones.
  • Ejecutar tu primer código simple.
  • Prepararte para usar Google Colab.

Contenido clave

Python es un lenguaje muy usado en IA porque es claro, flexible y tiene muchas bibliotecas útiles. Aquí no buscamos memorizar todo; buscamos entender lo suficiente para no sentir miedo.

La idea es que aprendas a leer y escribir instrucciones simples, porque eso te abrirá la puerta a los modelos y a los notebooks.

Ejemplo básico

nombre = "Armando" edad = 30 print(nombre) print(edad)

Práctica

Escribe un ejemplo con tu nombre, tu edad y una lista con 3 cosas que te gustan aprender.
